# zhangLab_Rotation

This project aims to build a pipeline that uses spatial transcriptomics data to infer cell-cell communcation-informed spatial niches. The cell-cell communication is determined using ligand-receptor relationships and is spatially constraint with collective optimal transport*. 



* Cang, Z., Zhao, Y., Almet, A.A. et al. Screening cell–cell communication in spatial transcriptomics via collective optimal transport. Nat Methods 20, 218–228 (2023). https://doi.org/10.1038/s41592-022-01728-4
